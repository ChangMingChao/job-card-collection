# 岗位内推投递卡片合集

数据来源：腾讯文档「27届暑期实习&秋招提前批信息整理汇总表（欢迎分享到同学群，就业群）」。

本地文件：

- `index.html`：卡片式岗位合集，可搜索、按表页筛选、按有无内推筛选，并直达投递链接。
- `jobs-data.js`：页面使用的结构化岗位数据。
- `jobs.csv`：可直接用 Excel 打开的岗位数据备份。
- `extract_jobs.py`：重新抓取并生成数据文件的脚本。

当前抽取范围：

- 2027届内推暑期实习信息
- 27届非内推暑期实习信息
- 27届秋招提前批
- 27届日常实习
- 2026届春招内推

分类口径：

- 有内推码/内推链接：来源表页包含「内推」，或字段中提取到内推码/内推链接。
- 无内推码：来源表页为「非内推」或未提取到内推信息的岗位。
